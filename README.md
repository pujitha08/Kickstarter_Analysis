# Kickstarter Analysis

This project analyzes Kickstarter campaign data to identify patterns associated with campaign success and failure.

## Project Structure

- `data/raw/` → original datasets
- `data/processed/` → cleaned datasets
- `notebooks/` → Jupyter notebooks for analysis
- `src/` → reusable Python scripts
- `outputs/figures/` → plots and charts
- `outputs/tables/` → summary tables

## Goals

- Perform data cleaning and exploratory data analysis (EDA)
- Understand key drivers of campaign success
- Build predictive insights/models
